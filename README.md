# unsupervised-learning-pandas-numpy-sklearn

Practice project on unsupervised learning using Python and its' libraries.

Libraries used:
- Pandas
- Matplotlib.pyplot
- Numpy
- Seaborn
- Sklearn

Consists of:
- Data checks
- Plotting features (histograms and barplots)
- Feature engineering
- Data scaling (min-max and standardization)
- Agglomerative hierarchical clustering
- K-means clustering
- Visualizing the clustering results
